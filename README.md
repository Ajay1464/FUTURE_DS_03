This project analyzes user behavior across an e-commerce marketing funnel using Power BI. The dashboard tracks user actions from product views to cart additions and final purchases to understand conversion performance and drop-off points.

### Key Objectives
- Analyze user journey from View to Cart to Purchase
- Measure conversion rates across funnel stages
- Identify drop-off points in the customer journey
- Provide insights to improve marketing and sales performance

### Tools Used
- Power BI
- Data Cleaning and Transformation
- DAX Measures
- Funnel and Conversion Analysis

### Dashboard Highlights
- View to Cart Conversion Rate
- Cart to Purchase Conversion Rate
- Overall Conversion Rate
- Drop-Off Rate Analysis
- User Action Distribution

This project demonstrates how data analytics can help businesses optimize customer conversion and improve marketing performance.
